# PROJECT DESCRIPTION
Mobile only PWA

## Name: Scholar Grid

## Description:

Scholar Grid Platform es una aplicación web colaborativa que centraliza, organiza y monetiza recursos educativos digitales. Está dirigida principalmente a estudiantes universitarios y creadores de contenido académico, ofreciendo un espacio seguro para compartir materiales, interactuar socialmente y gestionar tareas personales.
La plataforma aborda una necesidad real del ecosistema académico: la dispersión de materiales educativos de calidad en múltiples canales informales (grupos de WhatsApp, drives compartidos, repositorios sin curación) y la falta de una vía de ingresos justa para quienes generan dichos recursos.

## Objetivos Especificos

Implementar un sistema seguro de gestión de usuarios con autenticación multifactor y control de roles (estudiante, creador, moderador, administrador).
Diseñar un módulo de publicación y categorización de contenido educativo (archivos, enlaces, posts) con búsqueda eficiente y filtros avanzados.
Desarrollar un sistema de interacción social (comentarios, votos, reportes) que fomente la calidad del contenido mediante retroalimentación comunitaria.
Integrar una pasarela de pago certificada (Stripe o PayPal) para la monetización de contenido premium con cumplimiento PCI DSS.
Construir una To-Do List personal vinculada al perfil del usuario con recordatorios, etiquetas y seguimiento de progreso.
Implementar mecanismos de moderación de contenido y cumplimiento con normativas de propiedad intelectual (DMCA).
Asegurar la calidad del software mediante pruebas automatizadas, CI/CD y seguimiento de métricas conforme a ISO/IEC 25010.
